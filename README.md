# 走地盘预测
- 通过历史比赛数据分析 使用特征值进行聚类 (射正 射偏 危险进攻 盘口 赔率等等) 将足球比赛进行分类 =
- 通过历史数据进行回测 统计收益率 筛选收益率高的比赛类别
- 利用已经训练好的kmeans模型预测进行中的比赛 符合筛选出来的比赛类别进行推送
# web技术
- 将预测技术与网页结合 实时统计盈利率(模拟投注) 采用bet365标准赔率 
- 统计长时间模拟投注的盈利率和命中率 
- 将实时预测结果公布在网站上 同时将历史预测进行公布
